# Efficient Neural Network Mapping Co-Design In Programmable Networks
We propose an hybrid neural network inference architecture that co-designs the mapping of a BNN across the data plane and control plane of a programmable network device. Through a teacher-student approach, a non quantized MLP is used to guide the dynamic retraining of the BNN.
